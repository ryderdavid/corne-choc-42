# Adjacent key map for Corne Choc column-stagger with MBK keycaps (QWERTY)
#
# Format: intended_letter -> letters you might accidentally hit
#
# Column stagger means vertical neighbors are offset slightly,
# and wide MBK caps increase lateral overlap. On a Corne 3x6:
#
#   Q W E R T   Y U I O P
#   A S D F G   H J K L ;
#   Z X C V B   N M , . /
#
# Each entry accounts for: same-row neighbors, column neighbors
# (above/below with stagger offset), and diagonal fat-fingers.

r -> etf

y -> tuh

i -> uok
o -> ipl
p -> o
